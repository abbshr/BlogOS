Blog-OS

这是一个基于

Node.js平台 + express框架 + mongoDb数据库的多人博客系统/社交网络

功能描述：

  • 多用户
  • 实时（RealTime）
  • 个人控制
    • 编辑个人信息
    • 添加头像
    • pv统计
    • 管理自己发布的文章
  • 标签
  • 分页
  • markdown发布文章
  • 归档
  • markdown留言
  • markdown评论
  • 可视化文本编辑器
  • 文章检索
  • 管理员后台

更新日期： 2013.6.9

你可以选择Quick Start参照例子自己创建blogos：

  • 生成一个目录 
    mkdir myblog
  • 安装express模块（在此之前你需要安装Node.js） 
    npm install -g express

  • 建立express应用框架 
    express -e ejs blogos

  • 更新依赖模块 
    cd blog&&npm install

或者直接使用这个例子

  • 进入mongodb/bin，开启mongoDb数据库 
    cd mongodb/bin
mongod -dbpath "/home/username/myblog/datebase"
  • 启动应用 
    cd myblog/blogos
node app.js
